# תוכנית פעולה — Pipeline תמלול שיעורים

## חלק 1 — הורדת ההקלטה
מוריד את קובץ ההקלטה האחרון מתיקיית Google Drive לתיקייה מקומית.

## חלק 2 — חיתוך (אם צריך)
אם הקובץ גדול (שיעור שעה = בדרך כלל 50-100MB), חותך אותו לחלקים קטנים —
כי כלי התמלול מקבל עד 25MB בכל פעם.

## חלק 3 — תמלול
שולח כל חלק ל-Whisper (OpenAI) ומקבל בחזרה טקסט מתומלל בעברית.

## חלק 3ב — שמירת התמלול הגולמי
שומר את התמלול המלא בנתיב הקבוע `master-library/transcripts/lesson-NN.md`.
זה קורה **לפני** שלב העיבוד — כדי שגם אם משהו נכשל בהמשך, התמלול לא הולך לאיבוד.

## חלק 4 — עיבוד לתובנות (כאן ה"מוח" של הpipeline)
שולח את התמלול הגולמי למודל AI שמחלץ ממנו תובנות מסודרות —
מה נלמד בשיעור, בפורמט שהסוכן האישי שלך (BOM Manager) יכול לקרוא.

## חלק 5 — עדכון מסמך הידע
מוסיף (לא מוחק את הקודם!) את התובנות החדשות ל-`master-library/master-course-lessons.md`.

---

## מערכות שצריך לחבר
- **Google Drive** — כדי להוריד את ההקלטות (יש MCP מחובר אצלך ✅)
- **OpenAI (Whisper + GPT)** — כדי לתמלל ולעבד (נדרש API key — SideQuest קצר בשלב הבא)

## סדר הפעולות
דרייב → הורדה → (חיתוך אם צריך) → תמלול → שמירת תמלול גולמי → עיבוד → עדכון מסמך ידע.
כרגע הכל ידני — מריצים פקודה בטרמינל, ורואים תוצאה.
